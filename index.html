Anti-Ransomware Security Solution for MS Windows

System Protection Against Viruses & Ransomware

1. Keep Your System Updated

  • Always install the latest Windows/macOS/Linux updates.
  • Enable automatic updates for your operating system and software.
  • Outdated software often has security loopholes that attackers exploit.

2. Use a Good Antivirus / Anti-Malware Tool

  • Install a reliable antivirus (e.g., Windows Defender, Kaspersky, Bitdefender, or ESET).
  • Keep the antivirus updated regularly.
  • Enable real-time protection to block threats instantly.
  • Use an anti-ransomware feature if available.

3. Be Careful with Downloads and Email Attachments

  • Do not download cracked software, pirated games, or unknown apps.
  • Always download software from official websites.
  • Do not open suspicious email attachments or click unknown links.
  • Verify the sender’s email address before downloading anything.

4. Use a Firewall

  • Keep the Windows Firewall or any third-party firewall enabled.
  • A firewall blocks unauthorized access and suspicious network activity.

5. Enable Real-Time Ransomware Protection

In Windows 10/11, go to:

  • Settings → Update & Security → Windows Security → Virus & Threat Protection → Ransomware Protection
  • Turn on Controlled Folder Access to protect important files.

6. Regular Backups

  • Maintain a backup of your important files.
  • Use external drives or cloud storage (Google Drive, OneDrive, Dropbox, etc.).
  • Prefer incremental backups so your data stays updated.
  • Disconnect the backup drive when not in use to protect it from ransomware.

7. Practice Safe Browsing

  • Avoid visiting unsafe websites or clicking on ads/pop-ups.
  • Use a secure browser (e.g., Chrome, Edge, Brave, or Firefox).
  • Use HTTPS-only sites for transactions.

8. Keep USB and External Devices Safe

  • Always scan USB drives before opening them.
  • Disable AutoRun for USB devices to prevent automatic malware execution.

9. Use Strong Passwords and 2FA

  • Use unique, strong passwords for each account.
  • Enable two-factor authentication (2FA) wherever possible.

10. Monitor Your System

  • Use tools like Process Explorer or Task Manager to check unknown processes.
  • Check your startup programs regularly.
  • If your system slows down or shows unusual pop-ups, run a full scan immediately.

11. If You Suspect Ransomware

  • Disconnect from the internet to prevent spreading.
  • Run a full scan using a trusted antivirus.
  • Use tools like Malwarebytes or Kaspersky Anti-Ransomware Tool.
  • Restore data from a clean backup if files are encrypted.

App Usage Guide

  1. You can backup your important files and folders. When you choose a file or folder for backup, the tool will ask you to log in to Google Drive. After login, your selected files will be backed up automatically.
  2. The Anti-Ransomware tool runs in the background that detects entropy and encryption of files. Additionally, ClamAV scans manually and runs in background also for extra protection.
  3. You can also use the Real-Time button in Backup Data to select a folder that you want the app to monitor. The folder will first be uploaded, and after that, any new file added will automatically be uploaded to Google Drive. After selecting your backup folder in Real-Time, you also need to restart your PC.
  4. Choose the Scan button to scan your files manually.
  5. In the Threat button, when you click on Threat History, you can view the results of the scanned files.
  6. You can also use the Language Setting (Switch Language) button to change the app language (English or Urdu).
  7. On the Main Page, there are two buttons: Check for Update and About.
Tip: Regularly update the app for maximum security and protection.
